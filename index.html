Apps Script Web App URL

This is the URL from your deployed Google Apps Script Web App. To update it, edit SCRIPT_URL at the top of index.html and re-upload to GitHub.

How to Deploy Apps Script
  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete the default code, paste the contents of apps-script.gs
  3. Save (Ctrl+S), then click Deploy → New Deployment
  4. Type: Web App | Execute as: Me | Access: Anyone
  5. Click Deploy, authorize, copy the Web App URL
  6. Paste the URL as SCRIPT_URL in index.html and push to GitHub
